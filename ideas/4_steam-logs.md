# Steam Logs — Human-Readable Diagnostics

## Problem

Users do not receive clear explanations for launch failures.

## Current State

Errors are either missing or presented as low-level technical logs.

## Proposed Solution

- Introduce centralized, human-readable logs (accessible via UI, e.g., "Logs" button):
  - clear error descriptions  
  - suggested fixes  
- Example:
  - incompatible Proton version  
- Add automated fix options:
  - switch Proton version  
  - adjust launch parameters  

## Expected Impact

- Reduced support load  
- Better user experience  
- Increased system transparency
