# Devices — Hardware Ecosystem Expansion

## Problem

The Steam hardware ecosystem is limited.

## Current State

Primary focus is on Steam Deck.

## Proposed Solution

- Expand hardware lineup:
  - peripherals:
    - keyboards (including ergonomic / Steam Deck-focused)
    - mice
    - controllers
    - headsets  
  - accessories:
    - cases  
    - mats  
    - Steam Machine components  

## Expected Impact

- Stronger ecosystem  
- Better user experience  
- Increased engagement
