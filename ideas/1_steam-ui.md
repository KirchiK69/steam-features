# Steam UI — Wide-Screen Utilization

## Problem

The Steam interface does not efficiently use available space on wide screens, leaving large unused areas on both sides.

## Current State

Core content is centered, while secondary blocks are stacked vertically, increasing the need for scrolling.

## Proposed Solution

- Redistribute content horizontally  
- Introduce a side panel with:
  - top sellers
  - new & trending
  - recommendations  
- Add adaptive breakpoints for wide displays  

## Expected Impact

- Higher content density  
- Reduced vertical scrolling  
- Better utilization of modern displays  
- Improved overall user experience
