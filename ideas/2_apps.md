# Apps Page — Enhanced Game Information

## Problem

Users struggle to evaluate how a game will perform on their specific hardware.

## Current State

Compatibility data is limited and fragmented, especially for Linux / SteamOS users.

## Proposed Solution

- Add a compatibility section:
  - Steam Deck status based on user data (similar to ProtonDB)
- Extend the review system with:
  - devices used to run the game and playtime per device
  - launch configuration:
    - Proton version
    - launch options
    - compatibility settings

## Expected Impact

- Better purchase decisions  
- Reduced refund rates  
- Improved experience for Linux / SteamOS users
