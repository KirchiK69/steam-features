# Compatibility Window — Launch Data Collection

## Problem

There is no centralized way to collect structured data about game launches across different configurations.

## Current State

User launch experiences are not aggregated or used systematically.

## Proposed Solution

- Introduce a first-launch dialog:
  - request user consent for data collection  
- Collect:
  - device configuration  
  - Proton version  
  - launch result (success / failure)  
- Provide flexible privacy controls:
  - selectable data categories  
  - full opt-out option  

## Expected Impact

- Improved compatibility data  
- More accurate recommendations  
- Faster issue detection
